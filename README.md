# relation-chart

- 家族关系图谱
- 人物关系图
- 电视剧小说人物关系

可以画出那种类似于复杂人物关系，各个人物基本都有交集没有中心的图，还有可以用不同颜色线表示不同关系

注意： 拖动请在白色背景图鼠标按住拖动

功能

1. 导入
2. 导出 png
